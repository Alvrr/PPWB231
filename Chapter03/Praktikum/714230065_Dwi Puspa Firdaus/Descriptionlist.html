singkatan dalam situs web

WWW
World Wide Web berisi halaman web, gambar, video, dan konten online lainnya yang dapat diakses melalui browser web.
HTTPS
Hypertext Transfer Protocol Secure memungkinkan pengguna situs web untuk mengirimkan data sensitif seperti nomor kartu kredit, informasi perbankan, dan kredensial login dengan aman melalui internet.
COM
Commercial (com) adalah ekstensi website yang menandakan bahwa perusahaan pengelolanya beroperasi di bidang komersial.
